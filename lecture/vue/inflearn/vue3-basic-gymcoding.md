## 2.주요 vscode plugin

| 항목                             | 설명                                                                                                                                                                                                                                                                                                                                   |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Indent-Rainbow**             | Tab 영역을 컬러별로 다르게 표시하여 코드라인이 길어 졌을때 읽기 편하게 합니다.                                                                                                                                                                                                                                                                                       |
| **Auto Rename Tag**            | HTML Tag에서 태그 이름을 바꾸면 쌍을 이루는 여는태그 또는 닫는태그 명도 함께 바꿔줍니다.                                                                                                                                                                                                                                                                               |

| **Live Server**                | HTML 파일 수정 시 새로고침 없이 바로바로 즉각 적용되도록 도와준다.                                                                                                                                                                                                                                                                                             |
